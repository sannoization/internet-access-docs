# Настройка WireGuard на Android и iOS

## Установка приложения
Откройте Google Play (Android) или App Store (iOS).
В строке поиска введите WireGuard или AmneziaWG
Установите приложение и откройте его.
👉 Ссылка на WireGuard для Android

👉 Ссылка на WireGuard для iOS

## Добавление туннеля через QR-код
В главном окне приложения нажмите на "+" (плюс) в правом нижнем углу.
Выберите "Сканировать QR-код".
Наведите камеру на предоставленный QR-код и отсканируйте его.
В поле "Имя туннеля" введите любое название (только латиницей, без пробелов, например: "VPN_Tunnel").
Нажмите "Сохранить".

## Добавление туннеля через файл конфигурации
В главном окне WireGuard нажмите "+" (плюс) в правом нижнем углу.
Выберите "Импорт из файла или архива".
Найдите файл конфигурации (он обычно заканчивается на .conf) и выберите его.
В поле "Имя туннеля" введите название (например, "MyVPN").
Нажмите "Сохранить".

## Подключение к VPN
В списке туннелей найдите созданное подключение.
Нажмите переключатель (тумблер) рядом с названием туннеля.
Если всё настроено правильно, статус изменится на "Подключено".

# Настройка WireGuard на Windows и macOS

## Установка приложения
Перейдите на официальный сайт WireGuard и скачайте версию для вашей операционной системы: Windows: WireGuard for Windows
macOS: WireGuard for macOS
Установите программу и запустите её.
📷 Окно WireGuard на Windows:

## Подключение через файл конфигурации 
В главном окне WireGuard нажмите "Добавить туннель" → "Импортировать из файла".
Выберите файл конфигурации (.conf).
В поле "Имя туннеля" укажите название (например, "WorkVPN").
Нажмите "Сохранить".
📷 Пример импорта файла:

## Подключение к VPN
Выберите добавленный туннель в списке.
Нажмите "Активировать".
Если подключение успешно, статус изменится на "Подключено".

## Отключение VPN
Чтобы отключиться от VPN, просто нажмите "Отключить" в приложении WireGuard.

💡 Важно: Если вы хотите, чтобы VPN автоматически подключался при запуске системы, можно включить "Запускать при старте" в настройках WireGuard.
