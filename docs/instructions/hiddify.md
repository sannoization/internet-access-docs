# 📱 Настройка Hiddify на Android и iOS
## 1.Загрузка и установка
Перейдите в Google Play Store или App store
В строке поиска введите "Hiddify Next". Также можно скачать отсюда
Нажмите "Установить" и дождитесь завершения установки.
### Запуск приложения
Откройте установленное приложение Hiddify Next.
## 2. Добавление профиля подключения
### Получение ссылки для подключения
Получите от вашего провайдера VPN ссылку подписки или QR-код для подключения.
### Добавление профиля
В главном окне приложения нажмите на кнопку "+" или "Добавить профиль".
Выберите способ добавления:
- Сканировать QR-код: если у вас есть QR-код, нажмите соответствующую кнопку и отсканируйте его.
- Импорт из буфера обмена: если у вас есть ссылка, скопируйте её, и приложение автоматически предложит импортировать профиль.
### Сохранение профиля
После успешного добавления профиля он появится в списке доступных подключений.
## 3. Подключение к VPN
### Выбор профиля:
В списке профилей выберите добавленный ранее профиль.
Установка соединения:
Нажмите на переключатель рядом с названием профиля, чтобы установить соединение.
При успешном подключении статус изменится на "Подключено".

# 💻 Настройка Hiddify на Windows и macOS
## 1. Установка приложения
### Загрузка
Перейдите на официальный сайт Hiddify Next и скачайте версию приложения для вашей операционной системы (Windows или macOS). Или лучше скачайте отсюда
### Установка
**Windows:** Запустите скачанный установочный файл и следуйте инструкциям мастера установки.
**macOS:** Откройте скачанный файл и переместите приложение в папку "Программы".
### Запуск приложения
Откройте Hiddify Next из меню "Пуск" (Windows) или из папки "Программы" (macOS).

- Импорт из буфера обмена: если у вас есть ссылка, скопируйте её, и приложение автоматически предложит импортировать профиль.
