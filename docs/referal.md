# Реферальная программа
Если приводите человека, то получаете 50% скидку на следующую оплату.

## Процесс
1. отписываетесь мне в лс, кого хотите привести, передаете человеку мой контакт
2. дальше этот человек должен мне написать по поводу подключения
3. я подключаю человека и даю ему доступ в чат техподдержки.
4. я прихожу к вам в личку и говорю, какие условия вы получаете на следующий сезон. Пункт 4 будет исполнен при условии, что друг по реферальной программе оплатил доступ в интернет.

## Правила
Человек, которого вы приводите, сможет воспользоваться скидку по реферальной программе, начиная со следующей оплаты.
